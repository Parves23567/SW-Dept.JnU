📖 অ্যাপ সম্পর্কে

জগন্নাথ বিশ্ববিদ্যালয় সমাজকর্ম বিভাগের ৬ষ্ঠ ব্যাচের (শিক্ষাবর্ষ ২০১৪-১৫) অ্যালামনাই ডিরেক্টরি। সবাই নিজের তথ্য আপডেট রাখতে পারবেন।

⚡ Google Sheets সংযোগ গাইড

  • ১. Google Sheets খুলুন → নতুন sheet তৈরি করুন
  • ২. প্রথম row-তে হেডার দিন: name, roll, mobile, email, father, district, job, institution, photo, facebook, smriti, result
  • ৩. Extensions → Apps Script → নতুন script তৈরি করুন
  • ৪. doGet() ফাংশন দিয়ে JSON API বানান
  • ৫. Deploy → Web App → Anyone can access
  • ৬. URL কপি করে নিচে SHEET_URL এ বসান

✏️ তথ্য যোগ করার নিয়ম

  • নিচের + বোতাম চাপুন
  • ফর্ম পূরণ করুন ও সাবমিট করুন
  • Google Sheets এ সরাসরি যোগ হবে
  • সবাই রিয়েল-টাইমে দেখতে পাবে
